# Быстрый старт

## Запуск приложения

1. Установите зависимости (если еще не установлены):
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте браузер по адресу `http://localhost:5173`

## Важно перед использованием

### 1. Настройте адрес мастер-контракта

Откройте `src/config.ts` и замените `MASTER_ADDRESS` на реальный адрес мастер-контракта BCL:

```typescript
MASTER_ADDRESS: "EQD4FPq-PRDieyQKkO5Fm8Tnsg6p4qN1kP5J5N5J5N5J5N5J5N", // Замените на реальный
```

### 2. Обновите манифест TonConnect

Отредактируйте `public/tonconnect-manifest.json`:

```json
{
  "url": "http://localhost:5173",
  "name": "TonFun Launchpad",
  "iconUrl": "http://localhost:5173/icon.png"
}
```

## Использование

1. **Подключение кошелька**: Нажмите "Connect Wallet" в шапке
2. **Создание токена**: 
   - Перейдите на страницу "Create New Token"
   - Заполните форму (имя, символ, описание, изображение)
   - Нажмите "Create Token"
   - Подтвердите транзакцию в кошельке
3. **Просмотр токенов**: На главной странице отображается список всех токенов
4. **Торговля**: 
   - Откройте страницу токена
   - Введите сумму для покупки/продажи
   - Нажмите "Buy" или "Sell"
   - Подтвердите транзакцию

## Возможные проблемы

- **Ошибка подключения кошелька**: Убедитесь, что у вас установлен кошелек TON (Tonkeeper, TON Wallet)
- **Ошибка создания токена**: Проверьте, что у вас достаточно TON для оплаты комиссии
- **Токены не отображаются**: Проверьте настройки API endpoints в `src/config.ts`

## Сборка для production

```bash
npm run build
```

Собранные файлы будут в папке `dist/`
